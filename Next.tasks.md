## Sprint 2 — 認証とフリーミアムのアクセス制御（2週間）
**ゴール: 無料は誰でも見られ、有料コンテンツは契約者だけが見られる状態にする**

> フリーミアム前提（service-plan.md §5）。無料ゾーンは登録不要 or メール登録、有料ゾーンはログイン必須。

- [ ] 認証基盤の選定（Supabase Auth / Auth0 / Clerk など）
- [ ] メール登録（Freeリード獲得）→ ログイン・セッション管理
- [ ] ロール/プラン設計（Free / Pro / Enterprise）
- [ ] 有料ゾーンの出し分け（`.locked` ぼかし→認証&プランで解除。lp-design.md §5）
- [ ] 有料データは認証後に fetch（クライアント埋め込みを避けソース流出防止。Sprint 1前提）
- [ ] Vercel の環境変数・シークレット管理（`.vercel`/`project.json`は引き続き非コミット）

**完了条件:** 無料ゾーンは未ログインで閲覧でき、有料ゾーンはPro以上でのみ解除される
