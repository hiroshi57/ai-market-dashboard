# サービス企画書 — AI未浸透市場ダッシュボード

> 最終更新: 2026-06-24 / ステータス: ドラフト（数値は仮説、レビューで確定）

---

## 1. 戦略の要旨

ダッシュボードを「入口」、業種特化AI SaaSを「本命」とする **2層・段階的** な事業構造をとる。
ダッシュボードで見込み客を集め・信頼を獲得し、そこからAI SaaS導入支援の商談へつなげる。

```
[L1] 市場インテリジェンス・ダッシュボード（フリーミアムSaaS）
        ↓ リード獲得・信頼構築
[L2] 業種特化AI SaaS 導入支援（介護 / 法律 / クリニック）  ← 本命の収益源
```

---

## 2. プロダクト2層構造

### L1: ダッシュボード（入口・フリーミアムSaaS）
- 提供物: AI未浸透市場の可視化（15業界の浸透率・優先度・ROI・TimesFM予測・機会マトリクス）。
- 役割: 集客とリード化。ダッシュボード自体でも有料転換で収益を出す。
- 課金: フリーミアム（後述）。

### L2: 業種特化AI SaaS 導入支援（本命）
- 提供物: `TOP_PICKS` に設計済みの介護/法律/クリニック向けAI導入パッケージ。
  - 介護: 音声介護記録AI、AIケアプラン生成 ほか（単施設ROI ¥2,400万/年）
  - 法律: 契約書レビューAI（RAG）、AI判例検索 ほか（事務所ROI ¥3,600万/年）
  - クリニック: 診察音声AIカルテ、レセプトAIチェック ほか（ROI ¥1,200万/年）
- 役割: 高単価・継続課金の本命。L1の閲覧データから商談を創出。

---

## 3. ターゲット顧客

| 層 | 顧客 | ニーズ |
|---|---|---|
| L1 | 新規事業企画担当（事業会社） | どの市場に参入すべきか発見したい |
| L1 | コンサルティングファーム | クライアント提案の根拠データ |
| L1 | VC / 投資家 | 投資先・市場機会の評価 |
| L1 | 地方銀行・信金（地域DX） | 取引先支援・地域産業のAI化機会 |
| L2 | 介護/法律/クリニック事業者 | 実際のAI導入（自社業務効率化） |

---

## 4. 提供価値

「AI未浸透市場の **発見 → 参入優先度 → ROI試算 → 実導入** までを一気通貫で支援する」

- 発見: 浸透率・密度データで「まだ空いている市場」を提示
- 優先度: 緊急/早急/余裕の参入タイミング判定（FORECAST_TABLE）
- ROI: 業種別・サービス別のROI試算（ROI_TABLE）
- 実導入: L2で具体的なAI SaaSパッケージとロードマップを提供

## 5. フリーミアム境界（要確定）

| コンテンツ | Free | Pro | Enterprise |
|---|:---:|:---:|:---:|
| 概要・KPIヒーロー | ✓ | ✓ | ✓ |
| 業界カード（浸透率/優先度の要約） | ✓ | ✓ | ✓ |
| 機会マトリクス（バブルチャート） | ✓ | ✓ | ✓ |
| 詳細ROIテーブル（ROI_TABLE 全項目） | — | ✓ | ✓ |
| TimesFM予測の具体数値（FORECAST_TABLE） | — | ✓ | ✓ |
| TOP_PICKS 完全版（ロードマップ/ARR/KSF） | 一部 | ✓ | ✓ |
| レポート出力（PDF/Excel） | — | ✓ | ✓ |
| データ更新通知 | — | ✓ | ✓ |
| 複数席・SSO・カスタムレポート | — | — | ✓ |
| L2 導入相談（優先サポート） | 問い合わせ | 問い合わせ | 専任 |

> ⚠ **要確定**: 「TOP_PICKS の無料で見せる範囲」をどこまでにするか（ぼかしプレビューの粒度）。
> L2の商談を生むには“さわり”は見せ、核心（具体KSF・収益モデル）は有料/相談に置くのが定石。

## 6. 価格仮説（要確定・レンジ）

| プラン | 価格レンジ（仮） | 想定 |
|---|---|---|
| Free | ¥0 | メール登録で閲覧。リード化が目的 |
| Pro | ¥9,800〜29,800/月 | 個人コンサル/企画担当 |
| Enterprise | ¥30万〜/年（複数席） | ファーム/金融機関/事業会社 |

> ⚠ **要確定**: 競合（市場調査レポート: 数十万〜/本、業界DB: 年額制）との相対価格で再検討。
> L2は別建て（AI SaaS本体の月額 + 導入支援フィー）。本ダッシュボードの価格はあくまで入口。

---

## 7. 競合・データ出典

- 競合/代替: 既存の市場調査レポート（矢野経済・富士キメラ等）、業界DB、コンサルの内製分析。
  - 差別化: 「AI浸透“率”という独自切り口 × ROI試算 × 実導入(L2)まで繋がる」点。
- データ出典: e-Stat（産業分類・従事者数）等の公的統計を想定。
  - ⚠ **要確定（リスク）**: e-Stat等の **二次利用・再配布のライセンス可否**。有料提供する場合は出典明記＋利用規約での取り扱いを要確認。TimesFM予測値の算出根拠も開示方針を決める。

---

## 8. KPI（ファネル）

```
登録数（Free）
  → 有料転換率（Free→Pro/Ent）
    → L2商談化率（有料/閲覧者→AI導入相談）
      → ARR（L1課金 + L2 SaaS）
```

- 先行指標: 登録数、アクティブ閲覧、有料コンテンツのロック解除クリック率。
- 結果指標: MRR/ARR、L2商談数・受注額。
