# LP / CTA 設計 — ダッシュボードを商談へつなぐ導線

> 最終更新: 2026-06-24 / ステータス: ドラフト
> 目的: 訪問者を「無料閲覧 → 有料転換 → L2（AI SaaS導入相談）」へ運ぶ導線を設計する。

---

## 1. 設計方針

- 既存の単一HTML（`ai-market-dashboard.html` / `index.html`）の構造を活かし、新規CSSは最小化。
- 流用する既存要素:
  - `.topbar`（483行〜）: 右側にCTAボタンを常設追加。
  - `.hero-block#sec-overview`（574行〜）: 価値訴求とメインCTAの場に。
  - `scrollToSection(id, el)`（2115行〜）/ `.on` パターン: アンカー導線に流用。
- フリーミアム境界は [service-plan.md](service-plan.md) の §5 に従う。

## 5. ぼかし（有料ロック）UIの実装メモ

- 有料コンテンツ要素に `.locked` クラス → `filter: blur(6px); pointer-events:none;` を付与。
- 上にオーバーレイ（半透明 + 鍵アイコン + アップグレードCTA）を絶対配置。
- 認証/課金状態（Sprint 2/3）に応じて `.locked` を付け外し。
  - 例: ログイン済み&Pro → `document.querySelectorAll('.locked').forEach(el=>el.classList.remove('locked'))`
- Sprint 1 でデータがJSON化されれば「無料データのみ初期ロード、有料データは認証後にfetch」が可能になり、ソース流出も防げる（クライアント埋め込みより安全）。
